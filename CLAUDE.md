## Design Context

Strategic + visual design context for this project lives in two root files (generated via the
`impeccable` skill):

- **[PRODUCT.md](PRODUCT.md)** — register (`product`), users, purpose, brand personality
  (calm · precise · trustworthy), anti-references (enterprise-bloat / generic AI-SaaS), design
  principles, and the WCAG 2.2 AA accessibility target.
- **[DESIGN.md](DESIGN.md)** — visual system: color palette, typography, elevation, component
  vocabulary, and Do's & Don'ts. Read this before building or changing any UI so new screens stay
  on-brand.
